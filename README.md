### Tools

 - Vite

## How To Install & Run 
Step 1: Clone project 
Step 2: Locate to Project folder\
Step 3: Run command `npm i` on terminal\
Step 4: Run command `npm run dev` or `yarn dev`\
Step 5: To build, run command `npm run build`
